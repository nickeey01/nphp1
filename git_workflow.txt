# Git Workflow for This Project

1. Initialize git (if not already done):
   git init

2. Add all files:
   git add .

3. Commit your changes:
   git commit -m "Initial commit"

4. Add your GitHub remote (replace with your repo URL):
   git remote add origin https://github.com/your-username/your-repo.git

5. Push to GitHub (main branch):
   git push -u origin main

6. For subsequent changes:
   git add .
   git commit -m "Describe your change"
   git push

# Notes
- If you get a 403 error, use a GitHub personal access token for authentication.
- Make sure you are pushing from the project root directory.
- To change the remote:
   git remote set-url origin https://github.com/your-username/your-new-repo.git
